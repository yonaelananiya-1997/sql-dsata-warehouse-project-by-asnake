Modern Data Warehouse Project
Project Overview

This project demonstrates the implementation of a modern Data Warehouse using the Medallion Architecture (Bronze, Silver, and Gold layers). The objective is to transform raw operational data into high-quality analytical data for reporting and business intelligence.

Although the project is still in progress, it demonstrates the complete design approach and core implementation of a Data Warehouse solution.

Project Objectives
Build a multi-layer Data Warehouse architecture.
Implement ETL processes using SQL.
Perform data cleansing and transformation.
Design dimension and fact tables.
Apply Star Schema modeling.
Prepare data for reporting and analytics.
Practice industry-standard Data Warehouse development.
Architecture

Bronze Layer

Load raw source data.
Preserve original data without modification.

Silver Layer

Clean and validate data.
Remove duplicates.
Handle missing values.
Standardize formats.
Transform source data into trusted business data.

Gold Layer

Build analytical tables.
Create Fact tables.
Create Dimension tables.
Prepare data for dashboards and business reporting.
